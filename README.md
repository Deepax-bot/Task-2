Project Title: LearnHub – Task 2 (Multi-Page EdTech Platform)
Description:
LearnHub is a modern, responsive multi-page educational platform developed as part of Task 2. The project focuses on creating a high-performance user interface for students to explore tech courses like Web Development, Python, and Data Science. It emphasizes clean navigation, professional layout consistency across multiple pages, and interactive user feedback.

Key Features:

Multi-Page Architecture: Includes fully styled Home, About, and Contact pages managed through a unified CSS system.

Modern UI/UX: Features a sticky glassmorphism navigation bar with functional dropdown menus for service categories.

Interactive Form Validation: Custom JavaScript implementation to ensure user data integrity on the contact form before submission.

Responsive Design: Fully optimized for mobile, tablet, and desktop views using CSS Flexbox and media queries.

Professional Footer: A multi-column footer providing quick navigation, subscription capabilities, and social media integration.

Technical Stack:

Frontend: HTML5, CSS3 (Custom Variables, Flexbox)

Interactivity: JavaScript (DOM Manipulation, Event Handling)

Icons & Fonts: FontAwesome 6.0, Google Fonts (Poppins)
